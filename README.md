
# Benefits Finder Pro

This build merges:
- senior screening
- disability screening for younger adults
- Pennsylvania property tax and prescription programs
- saved profiles and caregiver mode
- scoring labels stronger than “very likely”

## Public tool / business direction

Start with:
1. Pennsylvania pilot
2. Senior centers, churches, disability groups
3. Free public link with caregiver mode
4. Simple intake + results + official links

Revenue ideas:
- sponsored local benefits help service
- white-label version for nonprofits
- premium admin dashboard for agencies
- grant-funded community access model
